# Algorithms

## Fixed Window Counter
Simple counter per time window. Resets at window boundaries.

## Sliding Window Log
Stores timestamps in a sorted set. Removes expired entries.

## Sliding Window Counter
Estimates current rate using previous and current window counters.

## Token Bucket
Tokens refill at a constant rate. Bursts allowed up to capacity.

## Leaky Bucket
Requests enter a queue and are processed at a fixed rate.
